Linkcare Health Services

Sistema Integrado de Telemedicina

Aplicación de Telemedicina orientada a la planificación de tareas para pacientes hecha en Ruby. Estos pacientes usan una app Android para tomarse medidas de dispositivos (como pulsioximetros, espirometros, glucometros, tensiometros, podometros,etc.) y también para contestar cuestionarios. La aplicación ayuda a los profesionales a gestionar/crear protocolos de intervención para casos de estudio. Sistema virtualizado en Amazon WS. Como Backend se usa un núcleo de WS SOAP hechos en PHP

Delta & Omega (Indra-BMB)

Sistema Integrado de Digitalización e Indexación Documental

Sistema integrado de digitalización/indexación/exportación de documentación para empresas. Con una parte de digitalización masiva usando Kofax VRS y OCR de Google (Indra SCAN). Otra parte de Indexación manual hecha en .NET (DELTA SERVER & CLIENTS) y otra parte de verificación y exportación de documentación con una aplicación web hecha en PHP (OMEGA)




Lotus (Indra-BMB)

Sistema Integrado de Gestión Previa Hipotecária

Multiples aplicaciones Lotus Notes para gestionar toda la documentación previa a las Hipotecas bancárias ( comunicación entre bancos, cajas, notarios, gestión documental). Sistema enlazado con todos los servicios de Lotus Domino (SameTime, Mensajería, Bases de Datos, Controles de Acceso ACL).








ArDoc (AGBAR)

Sistema Integrado de Digitalización e Indexación Documental

Multiples aplicaciones Lotus Notes para gestionar toda la documentación de las empresas del grupo AGBAR (Aguas de Barcelona, Sorea, Aguas Andinas, Applus+, ITVs, ACSA, Adeslas, etc.). Gestión de roles, permisos, ACLs, controles de versionado documental. Replicaciones locales. Triple diseño (HTML, BlackBerry y Lotus Notes)

Forum Universal Cultures 2004 BCN
